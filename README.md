# Brand Crawler для MerchantPoint.ru

Scrapy паук для сбора данных о торговых точках с сайта merchantpoint.ru.

## Описание

Данный проект реализует веб-скрапер для сбора информации об организациях и их торговых точках с ресурса merchantpoint.ru. Паук следует пользовательскому пути: главная страница → кнопка "Бренды" → выбор организации → вкладка "Торговые точки" → сбор данных с каждой точки.
